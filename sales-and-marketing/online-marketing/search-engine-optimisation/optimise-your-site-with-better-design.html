Search engine optimisation

Optimise your site with better design

Some of the best search engine optimisation (SEO) results can be achieved by optimising site design and content to utilise the techniques used by search engines to index web pages. Good site design makes it easier for search engines to index your site, and increases your chances of a higher ranking in the results pages. Basic SEO techniques include:

  • site accessibility
  • high quality content
  • links to and from other websites
  • use of relevant keywords

Make your site accessible

The most important part of making a website search-engine friendly is good design. Well-written, easy-to-navigate pages - with interesting information and well-designed e-commerce systems - will get repeat visitors as well as satisfied customers. You should consider the technologies to be used and whether they lend themselves to SEO. Although SEO can always be improved, it is harder to do so once the technology is in place and the website has already been built. If you use a content management system, you should make sure it produces SEO-friendly pages.

For more information, see our guide on best practice in web design.

More technical considerations include the use of friendly URLs, standards-compliant code and dynamically created XML sitemaps that can be used by search engines to index your website. Find out about standards-compliant code on the World Wide Web Consortium website - Opens in a new window. Some technologies, such as Adobe Flash, may add value to your website, eg through the use of video content or increased interactivity. However, this will need to be balanced with possible search indexing, accessibility and usability issues.

Have high quality content

Regularly updated, well-written, relevant content will improve your SEO. It also provides a good reason for others to link to your website. If you use social media, such as blogs, it provides an excellent way of keeping your content fresh and is well received by search engines. For more information, see our guide: Web 2.0: a guide for business.

Mobile SEO

Nowadays, mobile devices, such as smart phones, are commonly being used to access search results. Search engines will 'qualify' content based on how well it will render on mobile devices. Although your website will be viewable on mobile devices, a more effective mobile presence may require a separate mobile website.

When optimising for mobile, you should think about the nature of 'mobile 'search' and whether location-based keywords should be included in your key phrases. If you design a mobile website you should consider:

  • using CSS layouts (style sheets) to support cross platform compatibility
  • page layout and information hierarchy given limited screen space
  • the types of content that you use, eg the use of rich media will increase download times
  • the placement of any navigation and ease of use on mobile devices
  • optimising e-commerce functionality for mobile, eg shopping cart

For more information, see our guide on the essentials of mobile marketing and m-commerce.
