Check if you need to be certified to import organic goods

Introduction

Organic food production within Europe is strictly regulated, and some importers will need to be certified before they are allowed to import organic goods.

This interactive tool will help you to establish what you need to do to be able to legally import organic produce. Please note that this only applies to crop or livestock products, processed food and animal feed.
